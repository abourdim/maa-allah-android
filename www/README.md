# 🕌 With God — مع الله

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **مع الله — دراسات في الدعوة والداعية (With God: Studies in Dawah and the Caller)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 dawah cards with Quranic references, 8 caller qualities with problem/solution format, daily habit tracker with streaks, self-assessment quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes (Earth, Night, Oasis) with localStorage persistence
- Audiobook narrator with karaoke, voice/speed/pitch, loop, sleep timer
- Responsive design + print styles
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
